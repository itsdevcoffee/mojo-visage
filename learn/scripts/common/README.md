# Common Utilities

Shared scripts used across all blocks.

## Scripts

### `run_tests.sh`
Recursively runs all Mojo test files in the tests/ directory.

**Usage:**
```bash
pixi run test-all
# Or directly:
bash scripts/common/run_tests.sh
```
